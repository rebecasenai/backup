## 🚀 Tecnologias

- **Python 3.8+** - Linguagem principal
- **Flask** - Framework web
- **Firebase Firestore** - Banco de dados NoSQL
- **JWT** - Autenticação via tokens
- **Flask-CORS** - Compartilhamento de recursos entre origens
- **Flasgger** - Documentação Swagger
- **Python-dotenv** - Gerenciamento de variáveis de ambiente

## ✨ Funcionalidades

- ✅ Autenticação de administrador com JWT
- ✅ CRUD completo de alunos
- ✅ Busca de aluno por CPF
- ✅ Filtro de alunos por status (ativo/bloqueado)
- ✅ Alteração parcial de dados (PATCH)
- ✅ Geração automática de ID sequencial
- ✅ Timestamp automático de cadastro
- ✅ Documentação Swagger interativa
- ✅ Tratamento de erros global
- ✅ CORS configurado
